# 🕸️ MacroData Storyboard
### *Institutional Global Financial Intelligence Terminal*

**MacroData Storyboard** is an elite, open-source financial intelligence platform designed for institutional-grade market analysis. By unifying global data streams into a high-performance React Native interface, it provides traders, analysts, and developers with a single, algorithmic lens into the world's most critical economic drivers.

## 💎 Core Value Proposition

Unlike traditional fragmented data tools, **MacroData Storyboard** bridges the gap between raw data collection and strategic insight. It leverages the powerful **AKShare** engine to aggregate cross-region instruments—from US Bond Curves to Asian Equity Indices—delivering them through a low-latency API to a stunning, dark-mode terminal.

### 🚀 Key Features

*   **Global Equity Mastery:** Real-time tracking of S&P 500, NASDAQ, DAX 30, Nikkei 225, and FTSE 100.
*   **The Bond Link:** Institutional access to US 10-Year Treasury Yields, the ultimate gravity of global finance.
*   **Macro Economic Tracking:** Integrated modules for China GDP Growth, Fed Interest Rates, and Eurozone inflation metrics.
*   **Commodity Intelligence:** Precision mapping for Spot Gold (SGE) and Brent Crude Oil trends.
*   **Production-Ready Architecture:** A robust FastAPI backend deployed on Vercel with a high-end React Native frontend.

## 🛠️ Technology Stack

| Component | Technology | Rationale |
| :--- | :--- | :--- |
| **Frontend** | React Native (Expo) | Cross-platform (Web/Mobile) consistency and high-end UI responsiveness. |
| **Backend** | FastAPI (Python) | High-speed, asynchronous data orchestration and automatic documentation. |
| **Data Engine** | AKShare Core | A verified aggregator of global financial databases. |
| **Deployment** | Vercel | Seamless CI/CD and globally distributed serverless execution. |

## 📦 Installation & Setup

### 1. Backend Synchronization
```bash
cd backend
pip install -r ../requirements.txt
python main.py
```

### 2. Frontend Launch
```bash
cd frontend
npm install
npm run web
```
